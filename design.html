Model Deisgn

Deep Learning

기계학습의 한 종류로 써 인공 신경망으로 사람의 학습 형태를 본 따 구현한 심층 학습방법.
복잡한 비선형 관계의 데이터들을 학습해 판단 할 수 있다. Feed Forward, Back Propagation을 통해 학습한다.

MLP

Input layer 와 output layer 사이에 하나 이상의 hidden layer가 존재하는 신경망

CNN

Image data를 부분으로 나누어 이에 대응하는 값을 생성, 이를 이용하여 image의 특징점을 찾는 학습 방법

RNN


hidden node가 방향을 가진 edge로 연결되어 순환구조를 이루는 인공신경망의 한 종류

ENSEMBLE

작은 모델들의 합으로 큰 모델을 만들어 더 보완된 결과를 내기 위한 모델.
